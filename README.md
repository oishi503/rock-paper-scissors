Rock Paper Scissors Game

This is a simple Rock Paper Scissors game built using HTML, CSS, and JavaScript as a beginner frontend project.

The game allows a user to play against a computer opponent, with scores updating in real time based on game outcomes.

## Features
- Interactive user interface
- Random computer-generated moves
- Real-time score tracking
- Win, lose, and draw detection

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Play (https://oishi503.github.io/rock-paper-scissors/)
1. Open the game in a browser
2. Click on Rock, Paper, or Scissors
3. The computer makes a random choice
4. The result and score are displayed instantly
